# Market Pulse Weekly Snapshot 模板

> 用于每周在 Telegram / GitHub Releases / Newsletter 上发布的总结，延续「板块 → 结论 → 操作」结构。

## 1. 本周主线（Top Drivers）
- **Macro Regime**：<一句话总结 + 关键数据/事件>
- **Geopolitics**：<一句话总结 + 影响板块>
- **Liquidity / Rates**：<一句话总结>

## 2. 板块结论（Sector Conclusions）
重复每日 trade plan 的结构，但聚焦在「本周最重要的 3-4 个板块」，每个板块包含：

```
### 板块：Energy & Defense Hedge
- **结论**：中东风险溢价维持，油价区间 80-86。
- **驱动**：OPEC 指引 + 国防补单。
- **操作**：
  1. CVX — 持有 15%，油价 <82 加仓 1%。
  2. LMT — 持有 15%，若订单确认则至 17%。
```

## 3. 模拟盘表现（$100K Simulation P&L）
- 起始资金 / 当前权益
- 本周收益、月初以来收益
- Top contributors / detractors（配比图或表格）

## 4. 创作者共识亮点（Creator Consensus Highlights）
- 用 3-4 条要点描述：
  - 主题 + 情绪（Bullish / Bearish）
  - 引用创作者（Bloomberg TV, 美投讲美股 等）
  - 关联股票/ETF

## 5. 下周观察清单（Watchlist）
- 数据：CPI、NFP、FOMC 等
- 公司：GTC、重要财报
- 触发条件：若 event 发生/不发生 → 对组合影响

## 附件 / Links
- `daily-updates/reports/` 中最近 5 份日报
- `daily-updates/trade-plans/` 中最近 5 份操作卡片
- `daily-updates/positions/` 中最近快照

> 运行顺序建议：
> 1. 汇总本周所有 trade plan / portfolio 变动。
> 2. 用此模板写 Markdown，保存到 `reports/weekly/2026-03-07-weekly.md`。
> 3. 通过 GitHub Release 或 Telegram 推送。
